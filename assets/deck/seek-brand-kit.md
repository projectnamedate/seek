## Fonts

- **Headings**: Inter / System Sans-Serif, Black (900), uppercase, 2-4px letter spacing
- **Body**: Inter / System Sans-Serif, Bold (700)
- **Labels**: Inter / System Sans-Serif, Semi-Bold (600)

Gamma recommendation: Use **Inter** for both heading and body fonts.

## Brand Style Notes
- Dark mode only — near-black backgrounds with saturated cyan accents
- Inspired by Solana Mobile / Seeker device design language
- Glow effects on key UI elements (cyan shadow)
- Rounded corners: 16px standard, 24px modals
- Gradients: #61afbd -> #cfe6e4 (primary), #61afbd -> #95d2e6 (accent)
